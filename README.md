# G3P2 Save Editor

Electron + React 기반의 `창세기전3 파트2` 세이브 에디터입니다.

## 주의사항

세이브 파일 구조상 데이터가 중복으로 관리되기 때문에 편집한 정보가 사라질 수 있습니다.

- 전투 시작 : `챕터 데이터` 로 `전투 데이터` 생성
- 전투 종료 : `전투 데이터` 를 `챕터 데이터` 에 덮어씀

`챕터 데이터`에 저장되는 캐릭터의 종류는 정해져 있으며 해당 목록은 아래에 나와 있습니다.

## 사용 방법

1. index.html 을 실행
1. 세이브 파일을 드래그 & 드랍
1. 편집이 완료된 후 다운로드

## 지원 기능

### 공통

- 세이브 타입 표시: 연대표 / 챕터 / 전투
- 현재 에피소드 표시: 미선택 / 에피소드4 / 에피소드5
- 체크섬 검증

### 연대표 / 챕터 세이브

- 돈 변경
- 인벤토리 추가/삭제
- 파티 추가/삭제
- 군단 변경
- 장비 변경
- 공격 타입 / 무기 타입 / 무기 그림 타입

### 전투 세이브
- 군단 변경
- 장비 변경
- 공격 타입 / 무기 타입 / 무기 그림 타입
