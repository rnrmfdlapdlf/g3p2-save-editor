# G3P2 Save Editor

Electron + React 기반의 `창세기전3 파트2` 세이브 에디터이다.

## 현재 기능

- `G3P_II*.sav` 파일 열기
- 세이브 위치 표시: 연대표 / 챕터 / 전투
- 에피소드 표시: 미선택 / 에피소드4 / 에피소드5
- 체크섬 검증
- 에피소드4/에피소드5 돈 표시
- 에피소드4/에피소드5 파티 표시
- 돈 수정 후 체크섬을 갱신한 복사본 저장

## 확인된 세이브 위치

| 항목 | 오프셋 |
| --- | ---: |
| 에피소드4 돈 | `0x63F8` |
| 에피소드5 돈 | `0x74A8` |
| 에피소드4 파티 | `0x62F0` |
| 에피소드5 파티 | `0x73A0` |
| 세이브 위치 코드 | `0x0C` |
| 장면 ID 후보 | `0x08` |

세이브 위치 코드는 4바이트 보수값으로 읽는다.

| 코드 | 위치 |
| ---: | --- |
| 7 | 연대표 |
| 4 | 챕터 |
| 1 | 전투 |
