# 코드표와 외형 프리셋

## 외부 코드표

기존 에디터의 Data 폴더에서 다음 코드표를 확인했다.

```text
D:\Games\창세기전3 파트2\창세기전3파트2_에디터_GOOD_edit 1.1\Data
```

| 파일 | UI 항목 | 프로젝트 상수 |
| --- | --- | --- |
| `chrface.txt` | 얼굴 | `CHARACTER_FACE_OPTIONS` |
| `chrname.txt` | 이름 | `CHARACTER_NAME_OPTIONS` |
| `chrjob.txt` | 직업 | `CHARACTER_JOB_OPTIONS` |
| `chrvoc.txt` | 목소리 | `CHARACTER_VOICE_OPTIONS` |
| `chrbody.txt` | 모델링 | `CHARACTER_BODY_OPTIONS` |

무기 외형은 프로젝트 내부에서 다음 표로 관리한다.

| UI 항목 | 프로젝트 상수 |
| --- | --- |
| 무기 공격 타입 | `WEAPON_ATTACK_TYPE_OPTIONS` |
| 무기 타입 | `WEAPON_TYPE_OPTIONS` |
| 무기 그림 타입 | `WEAPON_PIC_OPTIONS` |

현재 목록에 없는 값이 세이브에서 발견되면 UI에는 `현재 저장값`으로 보존 표시한다.

## 외형 프리셋 구조

프리셋은 `src/renderer/App.tsx`의 `characterAppearancePresets`에 있다.
`code`는 중복되어도 되며, React 목록 식별에는 별도 `id`를 사용한다.

```ts
{
  id: "죠안",
  code: 221,
  name: "죠안",
  values: {
    face: 685,
    name: 475,
    job: 951,
    voice: 1,
    body: 338,
    weaponAttackType: 1,
    weaponType: 14,
    weaponPicType: 56
  }
}
```

프리셋 선택 시 위 8개 값이 현재 선택 캐릭터의 외형 탭 draft에 복사된다.
