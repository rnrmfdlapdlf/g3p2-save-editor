# 캐릭터 외형, 장비, 군단

## 챕터 캐릭터 레코드

챕터 캐릭터 데이터는 고정 레코드 25개로 확인됐다.

| 항목 | 값 |
| --- | --- |
| 시작 오프셋 | `0x0748` |
| 레코드 크기 | `0x03a4` |
| 레코드 수 | `25` |
| 캐릭터 코드 상대 오프셋 | `+0x0a4` |
| 군단 상대 오프셋 | `+0x0bc` |
| 장비 base 상대 오프셋 | `+0x0ec` |

현재 확인된 인덱스는 다음 순서다.

| index | 캐릭터 코드 | 이름 |
| --- | --- | --- |
| 0 | `236` | 네리사 |
| 1 | `237` | 에피소드4 데미안 |
| 2 | `427` | 에피소드5 데미안 |
| 3 | `238` | 에피소드4 디에네 |
| 4 | `222` | 란 |
| 5 | `240` | 루시엔 |
| 6 | `246` | 에피소드4 루크랜서드 |
| 7 | `484` | 에피소드5 루크랜서드 |
| 8 | `398` | 리엔 |
| 9 | `241` | 리차드 |
| 10 | `242` | 에피소드5 디에네 |
| 11 | `243` | 마리아 |
| 12 | `223` | 베라모드 |
| 13 | `247` | 엠블라 |

## 전투 출격 유닛 스캔

전투 탭은 챕터 파티 테이블을 쓰지 않고 실제 출격 유닛 레코드를 스캔한다.

| 항목 | 값 |
| --- | --- |
| 스캔 시작 | `0xe000` |
| 캐릭터 코드 | 후보 offset의 2바이트 역수값 |
| 장비 base | `캐릭터 코드 offset + 0x48` |
| 군단 | `캐릭터 코드 offset + 0x18` |

전투 유닛 후보는 다음 조건을 만족해야 한다.

- 캐릭터 코드가 `CHARACTER_NAMES`에 등록되어 있다.
- `offset + 0x48` 장비 블록이 범위 안에 있다.
- 장비 슬롯 값이 알려진 장비 코드이거나 0이다.
- 무기 공격 타입이 `1`, `6`, `1479` 중 하나다.
- 장비 슬롯 중 하나 이상이 0이 아니다.

일부 캐릭터는 빠른 확인을 위해 알려진 전투 장비 base를 먼저 검사한다.

| 캐릭터 코드 | 이름 | 전투 장비 base |
| --- | --- | --- |
| `219` | 살라딘 | `0x15409` |
| `222` | 란 | `0xf827` |
| `223` | 베라모드 | `0xfcfc` |
| `220` | 크리스티앙 | `0x15db3` |
| `221` | 죠안 | `0x158de` |

## 외형과 무기 외형

외형 값은 장비 base를 기준으로 역산해서 읽는다.
챕터와 전투 모두 `장비 base = 캐릭터 코드 offset + 0x48` 구조로 맞춰져 있다.

| UI 항목 | 상대 오프셋 | 설명 |
| --- | --- | --- |
| 이름 | `-0x46` | 첫 번째 이름 슬롯 |
| 이름 복사본 | `-0x44` | 저장 시 이름과 같은 값으로 쓴다. |
| 목소리 | `-0x42` | `chrvoc.txt` 코드 |
| 모델링 | `-0x40` | `chrbody.txt` 코드 |
| 얼굴 | `-0x3e` | `chrface.txt` 코드 |
| 직업 | `-0x3c` | `chrjob.txt` 코드 |
| 무기 공격 타입 | `-0x32` | `1`, `6`, `1479` 등이 확인됨 |
| 무기 그림 타입 | `-0x04` | 무기 그래픽 코드 |
| 무기 타입 | `-0x02` | 무기 분류 코드 |

## 장비 슬롯

장비 슬롯은 장비 base에서 2바이트 단위로 이어진다.

| 슬롯 | 상대 오프셋 |
| --- | --- |
| 무기 | `+0x00` |
| 방어구 | `+0x02` |
| 목걸이 | `+0x04` |
| 반지 | `+0x06` |
| 허리띠 | `+0x08` |
| 신발 | `+0x0a` |

장비 값 0은 장비 해제로 취급한다.
그 외 값은 `EQUIPMENT_OPTIONS`에 등록된 코드표와 대조한다.

## 군단

군단은 장비 탭에 표시한다.

| 범위 | 위치 | 크기 |
| --- | --- | --- |
| 챕터 | `recordBase + 0x0bc` | 4바이트 역수값 |
| 전투 | `characterCodeOffset + 0x18` | 2바이트 역수값 |

군단 코드표는 `MERCENARY_OPTIONS`에 정의되어 있다.
